primenum.c

The isPrime function checks if it is a prime and returns a value of truth (0 = no prime, 1 = prime).

As a parameter, it asks for the number to be checked.

Die Funktion isPrime prüft ob es sich um eine Primzahl handelt und gibt einen Wahrheitswert zurück(0 = Keine Primzahl; 1 = Primzahl).

Als Parameter, fordert sie die zu prüfende Zahl.

_Bool isPrime(int num)
{
    if(num == 0 || num == 1) return 0;
    for(int i = 2; i < num; i++)
        if(num % i == 0) return 0;
    return 1;
}
laylandnum.c

The isLayland function checks if it is a Leyland number and returns a truth value (0 = no Leyland number, 1 = Leyland number).

As a parameter, it asks for the number to be checked.

Die Funktion isLayland prüft ob es sich um eine Leyland Zahl handelt und gibt einen Wahrheitswert zurück(0 = Keine Leyland Zahl; 1 = Leyland Zahl).

Als Parameter, fordert sie die zu prüfende Zahl.

_Bool isLeyland(int num, _Bool op)
{
    for(int i = 2; i < num; i++)
        for(int j = 2; j < num; j++)
        {
            if(pow(i, j) + pow(j, i) == num)
            {
                if(op) printf("%d = %d^%d + %d^%d", num, i, j, j, i);
                return 1;
            }
        }
    return 0;
}
gcdea.c

The function eucalg (Euclidean algorithm) uses the Euclidean algorithm to calculate the gcd(greatest common divisor) and returns the result.

As a parameter, the function requires the two numbers at which the gcd should be calculated.

Die Funktion eucalg(Euclidean algorithm) berechnet mittels dem euklidischen Algorithmus den ggT(gröÃŸte gemeinsame Teiler) und gibt das Resultat zurück.

Als Parameter fordert die Funktion, die zwei Zahlen, bei den der ggT berechnet werden soll.

int eucalg(int a, int b)
{
    if(a == b || a == 0) return b;
    else if(b == 0) return a;

    while(a != b)
    {
        if(a > b) a -= b;
        else b -= a;
    }
    return a;
}
