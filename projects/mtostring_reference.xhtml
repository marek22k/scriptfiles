Functions

Function/Declaration Description Argument 1 Argument 2 Return
int mtostring(long, char **); Converts a whole signed number into a string. Disk space is requested with malloc. It is recommended, so that no memory leak occurs, free the memory again with free. The number to be converted; Type of number should be long. The address of the char pointer in which the result is to be stored. Returns 0 on failure, otherwise 1.

Example

#include <stdio.h>  /* printf */
#include <stdlib.h>  /* abort, free */
#include "mtostring.h"  /* mtostr */


int main()
{
    char * x;
    if(!mtostr(-1243, x))
        abort();

    printf("String: %s", x);

    free(x);
    return 0;
}
