Functions

Function/Declaration Description Argument 1 Argument 2 Return
MMEMORY_BR memnew(void **, MMEMORY_GET); calls memory to the address of the pointer in those who address of the memory area copied to be. the number of bytes how large the memory area supposed to be. Returns 0 on failure, otherwise 1.
MMEMORY_BR memdel(void **); is a storage area free. the address of the pointer, which the address of the storage area contains the released to be. Returns 0 on failure, otherwise 1.
MMEMORY_BR memedi(void **, MMEMORY_GET); change the size of a memory area the pointer, which the address of the to changes in memory area contains the new size in bytes Returns 0 on failure, otherwise 1.

Macros

Macro Description Value
MMEMORY_H None None
MMEMORY_ARR(num, type) calculates the number of bytes in an array of num elements and from the data type type. (num * sizeof(type))
MMEMORY_GET in this type be the number of bytes to the functions passed. size_t or unsigned int
MMEMORY_BR the return type a function. bool or _Bool or int

Example

#include <stdio.h>  /* printf */
#include "mmemory.h"  /* memnew, MMEMORY_ARR, memedi, memdel */

int main()
{
    void * ptr;
    int * iptr, i;

    memnew(&ptr, MMEMORY_ARR(1, int));

    iptr = (int *) ptr;
    * iptr = 4;

    memedi(&iptr, MMEMORY_ARR(3, int));

    iptr[1] = 5;
    iptr[2] = 6;

    for(i = 0; i < 3; i++)
        printf("%d", iptr[i]);

    memdel(&ptr);

    return 0;
}
